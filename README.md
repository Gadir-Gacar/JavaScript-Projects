# JavaScript-Projects

Welcome to my portfolio repository! This project highlights my transition from learning theoretical concepts to engineering live, interactive web platforms. It serves as a centralized hub showcasing the responsive applications built throughout the course.

## 🚀 Projects Overview

*   **Portfolio Website:** A tailored digital resume engineered with semantic layouts and personalized style guides.
*   **Tic-Tac-Toe Game:** A dynamic browser-based game utilizing advanced JavaScript state handling and logic.
*   **Interactive Calculator:** A modern calculator application featuring real-time DOM updates and error-handling arrays.
*   **Pizza Menu System:** A responsive interface tracking user inputs and calculating runtime pricing matrices.

---

## 🛠️ Core Stack

*   **HTML5:** Semantic architecture and content blueprints.
*   **CSS3:** Responsive typography, structural grid layouts, and smooth animations.
*   **JavaScript (ES6+):** Dynamic user interaction, event-driven DOM manipulation, and asynchronous workflows.

## 🎯 Key Learning Takeaways

1. **Applied Creation:** Bridged the gap between abstract syntax and deployment-ready software.
2. **Advanced Debugging:** Developed structural methodologies for troubleshooting logical and layout bottlenecks.
3. **Framework Readiness:** Cultivated the fundamental programming logic needed to transition smoothly into React, Vue, or Angular.
